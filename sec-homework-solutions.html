    Section 1.1 Homework Solutions

    This will be where I put in some homework problems and solutions.

    Subsection 1.1.1 Techniques of Integration

    Checkpoint 1.1.1. TI1 Substitution Method.

    (a)
    Explain and demonstrate how to find the anti-derivative of \(\renewcommand{\log}{\ln} f(x)=7 \, x e^{\left(6 \, x\right)}\text{.}\)
    (b)
    Explain and demonstrate how to evaluate the following definite integral:
    \begin{equation*} \int_{ 6 }^{ 7 } 7 \, x e^{\left(6 \, x\right)} dx. \end{equation*}
    Checkpoint 1.1.1 was taken from Calculus for Team-Based Inquiry Learning.

    Checkpoint 1.1.2. TI2 Integration by Parts.

    (a)
    Explain how to find the anti-derivative of \(f(x)=7 \, x \cos\left(2 \, x\right)\text{.}\)
    (b)
    Explain how to evaluate the following definite integral:
    \begin{equation*} \int_{ 5 }^{ 7 } 7 \, x \cos\left(2 \, x\right) dx. \end{equation*}

    Checkpoint 1.1.3. TI3 Integration of Trigonometry.

    (a)
    Use trigonometric identities and \((1-z)^2=1-2z+z^2\) to explain and demonstrate how to find \(\displaystyle \int 3 \, \cos\left(x\right)^{2} \sin\left(x\right)^{5} dx\text{.}\)
    (b)
    Explain and demonstrate how to use double-angle identities to rewrite \(\displaystyle \int 16 \, \cos\left(3 \, x\right)^{8} \sin\left(3 \, x\right)^{8} dx\text{.}\) (Do not solve the rewritten integral.)

    Checkpoint 1.1.4. TI4 Trigonometric Substitutino.

    Consider these given integrals of trignometric functions.
    • \(\displaystyle \int \tan^2\theta\,d\theta=\tan\theta-\theta+C\)
    • \(\displaystyle \int \sec\theta\,d\theta= \log|\sec\theta+\tan\theta|+C\)
    Use appropriate trignometric substitutions and the above integrals to find each of the following.
    (a)
    (i)
    \begin{equation*} \int \frac{1}{\sqrt{16 \, x^{2} + 81}}\,dx \end{equation*}
    (ii)
    \begin{equation*} \int \frac{\sqrt{9 \, x^{2} - 4}}{x}\,dx \end{equation*}
    (iii)
    \begin{equation*} \int \frac{49}{\sqrt{-49 \, x^{2} + 16}}\,dx \end{equation*}

    Checkpoint 1.1.5. TI5 Table of Integrals.

    Using a table of integrals, evaluate \(\renewcommand{\log}{\ln} \displaystyle\int -\frac{9}{\sqrt{9 \, x^{2} - 4}} dx\text{.}\)

    Checkpoint 1.1.6. TI6 Partial Fractions.

    (a)
    (i)
    Explain and demonstrate how to use partial fractions to decompose \(\frac{2 \, x - 101}{x^{2} + 3 \, x - 40}\text{.}\)
    (ii)
    Consider the partial fraction decomposition
    \begin{equation*} -\frac{4 \, x^{2} + 3 \, x + 150}{x^{3} + 2 \, x^{2} + 36 \, x + 72}=-\frac{3}{x^{2} + 36} - \frac{4}{x + 2}. \end{equation*}
    Use this decomposition to compute \(\displaystyle\int -\frac{4 \, x^{2} + 3 \, x + 150}{x^{3} + 2 \, x^{2} + 36 \, x + 72}dx\text{.}\)

    Checkpoint 1.1.7. TI8 Improper Integrals.

    Explain and demonstrate how to write each of the following improper integrals as a limit, and why this limit coverges or diverges.
    (a)
    (i)
    \(\displaystyle\int_{ -7 }^{ -6 } \frac{1}{{\left(x + 7\right)}^{2}} dx.\)
    (ii)
    \(\displaystyle\int_{ -1 }^{ +\infty } \frac{1}{{\left(x + 4\right)}^{\frac{5}{9}}} dx.\)
    (iii)
    \(\displaystyle\int_{ -1 }^{ +\infty } \frac{1}{{\left(x + 3\right)}^{2}} dx.\)
    (iv)
    \(\displaystyle\int_{ 9 }^{ 11 } \frac{1}{{\left(x - 9\right)}^{\frac{5}{9}}} dx.\)

    Subsection 1.1.2 Applications of Integration

    Checkpoint 1.1.8. AI1 Average Values.

    Explain and demonstrate how to compute the average value of \(f(x)=-15 \, x^{2} + 10 \, x - 1\) over the interval \([-4, 4]\text{.}\)

    Checkpoint 1.1.9. AI2 Arclength.

    Consider the curve \(y=\sqrt{x + 5}\) defined on \([1, 2]\text{.}\)
    (a)
    (i)
    Estimate the arclength of this curve with \(4\) line segments with \(\Delta x = 0.25\text{.}\)
    (ii)
    Find an integral that computes the arclength of this curve.

    Checkpoint 1.1.10. AI3 Volumes of Revolution.

    Consider the region bounded by \(y=\frac{5}{2} \, x, y=0, x=0, x=3\text{.}\)
    (a)
    (i)
    Find an integral which computes the volume of the solid formed by rotating this region about the \(x\)-axis.
    (ii)
    Find an integral which computes the volume of the solid formed by rotating this region about the \(y\)-axis.

    Subsection 1.1.3 Sequences and Series

    Checkpoint 1.1.11. SQ1 Sequences.

    (a)
    (i)
    Find the first 5 terms, beginning with \(a_{ 0 }\text{,}\) of the sequence defined by the closed formula \(a_n=2 \, \left(-4\right)^{n}\text{.}\)
    (ii)
    Find the first 5 terms of the sequence defined by the recursive formula \(a_{ 1 }= -2, a_{n+1}=a_{n} - 3\text{.}\)
    (iii)
    Find a closed form for the following sequence: \(3, -9, 27, -81, 243, \dots\)
    (iv)
    Find a recursive form for the following sequence: \(4, 9, 14, 19, 24, \dots\)

    Checkpoint 1.1.12. SQ2 Sequence Properties and Limits.

    For each of the following sequences, determine which of the properties: monotonic, bounded and convergent, the sequence satisfies. If a sequence is convergent, determine to what it converges.
    (a)
    (i)
    \(\displaystyle \left\{ \frac{4 \, n^{3}}{n^{2} + 2} \right\}_{n=0}^\infty\text{.}\)
    (ii)
    \(\displaystyle \left\{ \left(\frac{1}{6}\right)^{n} \right\}_{n=0}^\infty\text{.}\)

    Checkpoint 1.1.13. SQ3 Partial Sum Sequence.

    (a)
    (i)
    Consider the partial sum sequence \(\displaystyle A_n=\left(1\right)+\left(-5\right)+\left(25\right)+\cdots+ \left(1\cdot \left( -5\right)^n \right).\)
    (A)
    (A)
    Find a closed form for \(A_n\text{.}\)
    (B)
    Does \(\{A_n\}\) converge? If so, to what value?
    (ii)
    Consider the partial sum sequence \(\displaystyle B_n=\sum_{i=1}^n \left( \frac{1}{3 \, i + 2}-\frac{1}{3 \, i + 5} \right).\)
    (A)
    (A)
    Find a closed form for \(B_n\text{.}\)
    (B)
    Does \(\{B_n\}\) converge? If so, to what value?

    Checkpoint 1.1.14. SQ4 Geometric Series.

    Find the limit of the following series.
    \begin{equation*} 1 + \sum_{n = 1 }^\infty -3\cdot\left(-\frac{2}{3}\right)^n. \end{equation*}

    Checkpoint 1.1.15. SQ5 Basic Convergence Tests.

    For each of the following series, use the Divergence, Alternating Summation or Integral test to determine if the series converges.
    (a)
    (i)
    \(\displaystyle \sum_{n=1}^\infty \frac{5 \, n^{2} + 4 \, n}{n^{2}}.\)
    (ii)
    \(\displaystyle \sum_{n=1}^\infty \frac{1}{n^{2}}.\)
    (iii)
    \(\displaystyle \sum_{n=1}^\infty \frac{{\left(n^{4} + 3 \, n^{3}\right)} \left(-1\right)^{n}}{4 \, n^{5} + n^{4} + 4 \, n^{3} + 3 \, n^{2}}.\)

    Checkpoint 1.1.16. SQ6 Comparison Tests.

    For each of the following series, determine if it converges or diverges, and explain your choice.
    (a)
    (i)
    \(\renewcommand{\log}{\ln}\displaystyle \sum_{n= 3 }^\infty \frac{2}{2^{n} + 2}.\)
    (ii)
    \(\renewcommand{\log}{\ln} \displaystyle \sum_{n= 3 }^\infty \frac{3}{3 \, n^{2} + 4 \, n}.\)

    Checkpoint 1.1.17. SQ7 Ratio and Root Tests.

    Use the ratio or root test to determine if \(\displaystyle \sum_{n=1}^\infty \frac{n^{n}}{4^{n}}\) converges or diverges.

    Checkpoint 1.1.18. SQ8 Absolute Convergence.

    For each of the following series, determine if the series is convergent, and if the series is absolutely convergent.
    (a)
    (i)
    \(\displaystyle \sum_{n=1}^\infty \frac{5 \, \left(-1\right)^{n + 1} n}{2 \cdot 5^{n}}.\)
    (ii)
    \(\displaystyle \sum_{n=1}^\infty \frac{2 \, \left(-1\right)^{n + 1} n}{2 \, n^{2} + 3}.\)
